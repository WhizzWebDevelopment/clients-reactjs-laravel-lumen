* under backend folder
 $ composer install

 $ php artisan server

* under frontend folder:
 npm install
 npm start

then open in browser:
http://localhost:3000/

* lookup first or last name is starting as the input text

* MySQL database configuration see backend folder.env file:
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=college
DB_USERNAME=root
DB_PASSWORD=password

- database name is college; table name is clients.
- database file is under /Database-file/ folder, please import into "college" DB.
